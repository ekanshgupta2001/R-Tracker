# R-Tracker — FTC Driver Practice Suite

Built by **Team 27502 Rundle Robotics Castle** for the FTC 2025-2026 DECODE season.

Live site: *(link coming soon)*

---

## Features

- **TeleOp Practice** — drive a simulated mecanum or tank robot around the DECODE field using a gamepad or keyboard
- **Field-centric and robot-centric driving** — toggle between drive modes on the fly
- **Mecanum and tank drivetrains** — switch drivetrains and see realistic motor power mixing
- **Realistic physics** — momentum, acceleration, friction, input delay, and wall/obstacle collisions
- **DECODE field** — accurate field layout with collision zones for goals and ramps
- **Gamified driver training** — 12 levels across 4 difficulty tiers (Beginner, Intermediate, Advanced, Expert) with Gold/Silver/Bronze scoring, path corridors, ghost indicator, and countdown
- **Pedro Pathing-compatible path planner** — visual autonomous path builder with draggable waypoints, Bezier curve control points, and one-click Java code export
- **Light / dark mode** — toggle across all pages

---

## Tech Stack

Pure HTML, CSS, and JavaScript — no frameworks, no dependencies, no installation required. Every page is a single self-contained file.

## How to Use

**Option 1 — Live site:**
Visit the live site link above (coming soon).

**Option 2 — Run locally:**
```bash
git clone https://github.com/your-repo/ftc-driver-sim.git
cd ftc-driver-sim
open home.html
```
No build step. Just open `home.html` in any modern browser.
